# MOMO Order Tracking

A simple real-time order tracking app with separate customer and vendor views.

## Features
- Live order tracking updates
- Customer and vendor interfaces
- Out-of-stock toggle support
- MOUN10 branding and refreshed UI

## Requirements
- Node.js
- npm

## Installation
```bash
npm install
```

## Run locally
```bash
npm start
```

The app will be available at http://localhost:3000.
